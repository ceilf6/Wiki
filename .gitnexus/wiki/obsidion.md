## Module Overview

This module is a comprehensive knowledge base designed for academic study, focusing on exam-oriented learning. It organizes course materials, problem-solving approaches, and reference notes across multiple engineering and management disciplines. The core philosophy is to start from problems and align solution methods with instructor expectations.

The module serves as a centralized repository for:
- **Engineering Chemistry** (工程化学)
- **Digital Circuits** (数字电路)
- **Data Structures** (数据结构)
- **Quality Management** (质量管理)
- **Project Management** (项目管理)

## Content by Subject

### 1. Engineering Chemistry (工程化学)
Focuses on chemical reaction principles and applications:
- **Chapter 1**: Chemical reaction control and application
  - Thermodynamic systems and processes
  - Heat and work in chemical reactions
  - Hess's law and reaction enthalpy calculations
  - Gibbs free energy and reaction spontaneity
  - Chemical equilibrium and Le Chatelier's principle
- **Chapter 3.2**: Chemical bonding concepts

### 2. Digital Circuits (数字电路)
Covers fundamental digital electronics:
- **Chapter 1**: Digital circuit basics
  - Number systems and conversions
  - Binary codes (BCD, complement codes)
  - Logic operations (AND, OR, XOR, etc.)
- **Chapter 2**: Logic functions and combinational circuits
  - Boolean algebra rules
  - Karnaugh map simplification
  - Encoders and decoders
  - Adder circuits
- **Chapter 3**: Combinational logic circuit design
  - BCD adder design using 3→8 decoders
  - Correction logic for decimal addition
- **Chapter 4**: Additional topics (content pending)

### 3. Data Structures (数据结构)
Comprehensive coverage of data structures and algorithms:
- **Fundamentals**: Time/space complexity, storage structures
- **Linear Structures**: Arrays, linked lists, stacks, queues
- **Non-linear Structures**: Trees, graphs (adjacency lists)
- **Algorithms**: Recursion (with stack simulation), sorting (heap sort)
- **C++ Implementation**: Classes, templates, memory management
- **Special Topics**: String operations, sparse matrices, generalized lists

### 4. Quality Management (质量管理)
Introduces quality management principles:
- **Core Concept**: Quality is inversely proportional to variability
- **SPC (Statistical Process Control)**: Tools for monitoring and reducing process variation
- **Terminology**: Variability, variance, and related terms

### 5. Project Management (项目管理)
Covers project planning and scheduling:
- **PERT (Program Evaluation and Review Technique)**:
  - Task dependency analysis
  - Critical path identification
  - Time calculations (earliest/latest start/finish)
  - Float/slack time concepts
- **Online Tasks**: Sub-project (SP) breakdown for a smart waste sorting project
  - SP1-SP6 phases from requirements to delivery

## Key Concepts and Patterns

### Thermodynamic Notation
- **Δ (Delta)**: Represents change in quantities (ΔU, ΔH, ΔG)
- **Standard state**: 100 kPa pressure, 1 mol/L concentration, 298.15 K temperature

### Digital Circuit Design Flow
```
Functional requirements → Truth table → Karnaugh map simplification → Logic expression → Circuit diagram
```

### Data Structure Implementation Patterns
- **Sequential storage**: Arrays with O(1) access but fixed size
- **Linked storage**: Dynamic allocation with pointer overhead
- **Hybrid approaches**: Static linked lists using array indices

### Project Management Workflow
```
Task identification → Dependency mapping → PERT diagram → Critical path analysis → Schedule optimization
```
