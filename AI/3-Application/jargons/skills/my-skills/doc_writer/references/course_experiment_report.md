# 课程 / 实验报告模板

## 适用场景

适用于课程作业、实验报告、课程设计总结、实训报告、读书或学习任务总结等偏教学场景的正式文档。

## 使用要求

- 优先保证学术或教学文体清晰、完整。
- 不得编造实验结果、数据来源或结论。
- 若用户未提供真实数据，可保留“结果待补充”占位。
- 如果学校或老师指定了格式，应以指定格式优先。

## 标准结构

### 标题

[课程 / 实验名称]报告

### 基本信息

- 姓名：[待补充姓名]
- 学号：[待补充学号]
- 班级：[待补充班级]
- 课程名称：[待补充课程名称]
- 指导教师：[待补充教师姓名]
- 完成日期：[待确认日期]

### 一、摘要

概括报告主题、采用的方法、主要过程和最终结论。

示例写法：

本文围绕 [实验 / 课程任务主题] 展开，重点介绍了 [方法或过程]，并对 [结果或现象] 进行了分析。通过本次学习 / 实验，进一步理解了 [核心知识点]，达到了预期学习目标。

### 二、实验目的 / 学习目标

1. [目标 1]
2. [目标 2]
3. [目标 3]

### 三、实验环境 / 前置条件

若适用，可写硬件环境、软件环境、工具版本、实验材料等。

- 硬件环境：[待补充]
- 软件环境：[待补充]
- 实验材料：[待补充]

### 四、实验原理 / 理论基础

说明本次实验或课程任务涉及的基本概念、原理、公式、方法或理论依据。

### 五、实验过程 / 实施步骤

按时间或步骤顺序写清操作过程。

1. [步骤 1]
2. [步骤 2]
3. [步骤 3]

### 六、结果展示

说明实验现象、运行结果、输出截图、数据表格或阶段成果。

- 结果 1：[待补充]
- 结果 2：[待补充]
- 结果 3：[待补充]

### 七、结果分析

围绕结果是否符合预期、原因是什么、有哪些偏差、如何解释现象进行分析。

建议从以下角度展开：

- 是否达到预期目标
- 结果成因分析
- 存在问题与误差来源
- 结果的实际意义

### 八、结论与总结

总结本次实验 / 学习任务的主要收获、最终结论及不足之处。

### 九、改进建议

如适用，可补充后续优化方向、改进方法或继续研究的问题。

## 文风要求

- 以客观陈述为主，避免过度口语化。
- 对实验过程使用“首先、其次、最后”等清晰连接词。
- 对结论使用“结果表明”“分析可知”“综合来看”等正式表达。
