用户 → claude code -用户输入+系统提示词（包含技能）→ 模型 -决定使用技能→  JSON（按照初始化请求中告诉的格式去调用技能）→ claude code -读取技能全文（渐进式纰漏）→ 模型 → claude code → 用户（最后不再 tool_use 后才会返回 UI 给用户，这点在 cc 源码解析中提过）

初始化时在请求体中有 Tools 中有一个 skill 告诉了大模型该如何去使用技能，例如调用时要说明技能名称等等

在用户输入后会拼接 CLAUDE.md 文件以及 system-reminder系统提示词，在系统提示词中告诉了大模型本地有哪些技能可供使用，以及在什么时候调用技能
